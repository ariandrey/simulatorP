Kodda Kullanilan Onemli C Yapilari

switch-case → Kullanici hangi komutu yazdiysa dogru islemi calistirmak icin.

if-else → Avlanma sonucu, enerji yetersizligi, barinak bulma gibi kosul kontrolleri icin.

Ternary operator → Envanterde barinak durumunu kisa yoldan gostermek icin.

for loop → Tehlike dalgalarini (F komutu) arka arkaya calistirmak icin.

do-while → En az 1 kez calismasi gereken yapilar (parola girisi gibi).
